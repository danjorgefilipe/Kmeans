AI algorithm for related videos


Project Main Objective

The main objective of the project is, that given a user ID, you get in return a number of personalised videos for that specific user.


Architecture

The main language used in this project is going to be python, for its ease of use and avaiable tools to work with AI.

Besides this, we will be using Flask, Sql-lite, and Sklearn.


-- Flask

Flask is a “microframework” for Python, and is an excellent choice for building smaller applications, APIs, and web services.


-- sklearn

Scikit-learn is a free software machine learning library for the Python programming language. It features various classification, regression and clustering algorithms and is designed to interoperate with the Python numerical and scientific libraries NumPy and SciPy.

We will be using Kmeans as the clustering algorithm from sklearn, because it is faster than KNN ( closest Nearest Neibor ) in production.
